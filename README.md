## "직접 경험을 해야 지식이 된다."

혼자만 이해하고 끝나는 공부가 아니라, 제 글을 읽는 누군가와 함께 나누며 더 성장하는 공부를 하고 싶습니다.

같이 공부하는 사람들에게도 더 재미있게 전달되길 바랍니다.
