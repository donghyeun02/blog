# 불린 대수 기초

불린 대수는 논리게이트의 수학적 기반이 되는 대수학입니다. 이 글에서는 불린 대수의 기본 개념과 법칙들을 학습하고, 이를 논리게이트 설계에 어떻게 적용하는지 알아보겠습니다.

## 불린 대수란?

불린 대수는 1854년 조지 불(George Boole)이 창안한 수학적 체계로, 참(1)과 거짓(0) 두 가지 값만을 다루는 대수학입니다. 디지털 회로 설계의 이론적 기반이 되며, 논리게이트의 동작을 수학적으로 표현할 수 있게 해줍니다.

## 기본 연산

불린 대수에는 세 가지 기본 연산이 있습니다:

### 1. AND 연산 (논리곱)

- 기호: `·` 또는 생략
- 예시: `A · B` 또는 `AB`
- 모든 입력이 참일 때만 결과가 참

### 2. OR 연산 (논리합)

- 기호: `+`
- 예시: `A + B`
- 하나라도 입력이 참이면 결과가 참

### 3. NOT 연산 (논리부정)

- 기호: `'` 또는 `¬`
- 예시: `A'` 또는 `¬A`
- 입력을 반전

## 불린 대수의 기본 법칙

### 항등 법칙 (Identity Laws)

```
A + 0 = A
A · 1 = A
```

### 영원 법칙 (Null Laws)

```
A + 1 = 1
A · 0 = 0
```

### 멱등 법칙 (Idempotent Laws)

```
A + A = A
A · A = A
```

### 보완 법칙 (Complement Laws)

```
A + A' = 1
A · A' = 0
```

### 교환 법칙 (Commutative Laws)

```
A + B = B + A
A · B = B · A
```

### 결합 법칙 (Associative Laws)

```
(A + B) + C = A + (B + C)
(A · B) · C = A · (B · C)
```

### 분배 법칙 (Distributive Laws)

```
A · (B + C) = (A · B) + (A · C)
A + (B · C) = (A + B) · (A + C)
```

### 드모르간 법칙 (De Morgan's Laws)

```
(A + B)' = A' · B'
(A · B)' = A' + B'
```

### 흡수 법칙 (Absorption Laws)

```
A + (A · B) = A
A · (A + B) = A
```

## 논리게이트와의 관계

각 논리게이트는 불린 대수의 연산에 대응됩니다:

| 게이트 | 불린 표현       | 진리표                                             |
| ------ | --------------- | -------------------------------------------------- |
| AND    | `F = A · B`     | A=0,B=0→F=0; A=0,B=1→F=0; A=1,B=0→F=0; A=1,B=1→F=1 |
| OR     | `F = A + B`     | A=0,B=0→F=0; A=0,B=1→F=1; A=1,B=0→F=1; A=1,B=1→F=1 |
| NOT    | `F = A'`        | A=0→F=1; A=1→F=0                                   |
| NAND   | `F = (A · B)'`  | AND의 결과를 반전                                  |
| NOR    | `F = (A + B)'`  | OR의 결과를 반전                                   |
| XOR    | `F = A'B + AB'` | 입력이 다를 때만 참                                |

## 불린 함수의 간소화

복잡한 불린 함수를 간단하게 만드는 과정을 불린 함수의 간소화라고 합니다. 이를 통해 회로를 더 효율적으로 설계할 수 있습니다.

### 예제: 불린 함수 간소화

다음 함수를 간소화해보겠습니다:

```
F = A'B'C + A'BC + AB'C + ABC
```

**1단계: 공통 인수 추출**

```
F = A'B'C + A'BC + AB'C + ABC
  = A'(B'C + BC) + A(B'C + BC)
```

**2단계: 분배 법칙 적용**

```
F = (A' + A)(B'C + BC)
  = 1 · (B'C + BC)
  = B'C + BC
```

**3단계: 추가 간소화**

```
F = B'C + BC
  = C(B' + B)
  = C · 1
  = C
```

따라서 `F = C`로 간소화됩니다!

## 카르노 맵 (Karnaugh Map)

카르노 맵은 불린 함수를 시각적으로 간소화하는 도구입니다.

### 2변수 카르노 맵 예제

함수 `F = A'B + AB'`의 카르노 맵:

```
    B
A    0  1
0    0  1
1    1  0
```

이 경우 간소화할 수 없어서 원래 형태가 최소 형태입니다.
